# ManagementScience

# Topics
- Linear Models
 Min, Max, Equals
 "Make 2 widgets", Assignment, Scheduling, Flow, Shortest Path
- Nonlinear Models
- Decision Analysis
 Bayes
- Simulation Systems
 Simul8

- Solver (seems to be baked into Excel 2016)
- TreePlan (inexpensive education licenses available, ~$8 each for 12 students)
- Simul8
